# 🎯 Number Guessing Game (Python)

A simple command-line number guessing game built using Python.

The computer randomly selects a number between **1 and 100**, and the player keeps guessing until the correct number is found.

---

## 🚀 Features

- Random number generation
- User input
- Hints (Higher/Lower)
- Counts total attempts
- Beginner-friendly code

---

## 🛠️ Technologies Used

- Python 3
- random module

---

## ▶️ How to Run

```bash
python main.py
```
